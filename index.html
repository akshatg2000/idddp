Understanding mental development in children

Report Slides

Abstract

Monitoring mental development in children is crucial for ensuring proper functioning later in life, much like monitoring physical development. However, such monitoring is challenging and expensive, requiring administration of complex tests by trained professionals. This is particularly difficult for low-income countries like India and Malawi, resulting in a “detection gap” in these regions. The STREAM (Scalable Transdiagnostic Early Assessment of Mental Health) project aims to bridge this gap through a tablet-based assessment easily administered at home by non-specialist workers.

In this work, we first outline the techniques used to convert backend assessment data into meaningful features that capture a child’s performance, employing various computer vision and data analysis algorithms.

The main contribution of this work is to demonstrate how these features can be used to generate developmental scores across domains such as motor, social, and cognitive. We show how machine learning algorithms can predict these scores, comparable to those from expensive and time-consuming psychometric tests like GMDS and MDAT,by using these psychometric scores as training labels. Additionally, we employ Item Response Theory (IRT) to generate developmental scores directly from these features without relying on other test scores as training labels. This establishes that a tool like STREAM could be used to produce developmental scores, potentially becoming a standardized method for monitoring mental development.
